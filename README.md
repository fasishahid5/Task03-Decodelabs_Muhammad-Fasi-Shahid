#  General Knowledge Quiz – Python Project

A simple command-line quiz application built using Python.  
It uses control flow and data structures to ask multiple-choice questions and calculate the final score.

---

##  Features

- Multiple-choice questions
- Instant feedback (Correct / Wrong)
- Score tracking system
- Built using Python lists & dictionaries
- Simple and interactive CLI interface

---

##  Technologies Used

- Python 3
- Lists & Dictionaries
- Loops (for loop)
- Conditional statements (if/else)
- User input handling
